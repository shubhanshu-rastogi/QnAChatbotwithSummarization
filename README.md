# FinancialAssistant Workspace

This repository contains multiple related projects for document Q&A and summarization workflows.

## Projects

- `rag-session-qa/`
  - Session-based RAG document Q&A with FastAPI + React (Vite), ChromaDB, OpenAI embeddings/chat, and DeepEval evaluation.
  - See `rag-session-qa/README.md` for setup and usage.

- `TextSummarizationandQnAChatbot/`
  - Additional Q&A + summarization project variant.
  - See `TextSummarizationandQnAChatbot/README.macos.md` or `TextSummarizationandQnAChatbot/README.windows.md` for setup.

## Quick Start

Pick a project above and follow its README. Each project is self-contained with its own backend/frontend setup.
